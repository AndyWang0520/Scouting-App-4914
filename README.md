# Scouting-App-4914
Scouting App 4914
This is a mobile app for FIRST Robotics Competition (FRC) team 4914 to use for scouting during competitions. It allows team members to easily record and track data on other teams and their robots.


Installation
User must have java installed on the device version 7 or higher 
Download the rar file and launch the program

User guide
Enter the name of any specified team and increament their score according to the different catagories provided. 
Data can be exported and used for future refrence
