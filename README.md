# Scouting App 4914

This is a mobile app for FIRST Robotics Competition (FRC) team 4914 to use for scouting during competitions. It allows team members to easily record and track data on other teams and their robots.

## Installation

To install the app, follow these steps:

1. Ensure that you have Java version 7 or higher installed on your device. If you do not, you can download it from the [Java website](https://www.java.com/en/download/).
2. Download the `.jar` file from the [releases page](https://github.com/AndyWang0520/Scouting-App-4914/releases).
3. Launch the app by double-clicking the `.jar` file.

## User Guide

To use the app, follow these steps:

1. Enter the name of the team you want to scout.
2. Increment their score in the various categories provided.
3. Export the data for future reference.
